# Articles
Articles Imperial Task Angular with NodeJS and MySQL

1. Run mySql migration.

2. Run backend code:
    * npm install -g nodemon.
    * npm install express.
    * npm run dev.
  
3. Run frontend code:
    * npm install -g @angular/cli.
    * ng serve --open.

4. versions:
    * node 14.21.1.
    * angular 15.0.1.
    * npm 6.14.17.
