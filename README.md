# 📰 Articles Management System

A full-stack articles management application built with Angular, Node.js, and MySQL.
Developed as an Imperial Task project.

## ✨ Features

- 📋 **Article Listing** — View all articles in a table with title and publisher
- ➕ **Create Article** — Add new articles with title, content, publisher, and status
- 👁️ **View Article** — Full article details page
- ✏️ **Edit Article** — Update existing articles
- 🗑️ **Delete Article** — Remove articles
- 📌 **Article Status** — Published / Unpublished toggle

## 🛠️ Tech Stack

**Frontend:**
- Angular 15
- TypeScript
- HTML / CSS

**Backend:**
- Node.js
- Express.js
- MySQL

## 🚀 Getting Started

**Backend:**
```bash
cd be-nodejs
npm install
npm run dev
```

**Frontend:**
```bash
cd fe-angular
npm install -g @angular/cli
ng serve --open
```
